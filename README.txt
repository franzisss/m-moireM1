Ce dépôt Git contient la documentation du projet de recherche présenté dans le cadre du Master 1 "Humanités numériques et computationnelles" de l'Ecole des Chartes-PSL. 

Vous pouvez trouver :

1) dans le dossier intitulé "code", où on peut voir les trois cahiers de code ayant servi dans le mémoire : un  fichier pour la collecte des données, un fichier pour la production des scores d'intensité émotionnelle avec VADER, et un dernier fichier pour la production des graphiques.

2) dans le dossier intitulé "fichiers csv", on peut trouver les six dataframe avec les extraits de code et les scores attribués par VADER pour les textes français et anglais

3) une version en pdf du mémoire.
